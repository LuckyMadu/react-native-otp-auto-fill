# react-native-otp-auto-fill

Auto fill OTP library for React Native

## Installation

```sh
npm install react-native-otp-auto-fill
```

## Usage

```js
import OtpAutoFill from "react-native-otp-auto-fill";

// ...

const result = await OtpAutoFill.listen();
```
